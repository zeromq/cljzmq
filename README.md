# cljzmq

[![Build Status](https://travis-ci.org/zeromq/cljzmq.png)](https://travis-ci.org/zeromq/cljzmq)

Clojure bindings for ØMQ.

## Installation

Add the following dependency to your `project.clj` file:

```clojure
[org.zeromq/cljzmq "0.1.0-SNAPSHOT"]
```
